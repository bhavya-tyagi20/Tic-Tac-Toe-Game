# 🎮 Tic Tac Toe Game

## 📖 Description
A simple 2-player Tic Tac Toe game where users take turns marking X and O on a 3×3 grid. The game detects winning conditions and draw scenarios automatically.

## 🚀 Features
- Two-player gameplay  
- Winner detection  
- Draw detection  
- Reset game option  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

## ▶️ How to Run
Open `index.html` in a browser.
